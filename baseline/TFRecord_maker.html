TFRecord maker: from .jpg to .tfrecord¶

This notebook presents how to convert the image data from .jpg to .tfrecord. The main reference is KWOT SIN's blog

Arranging data according to classes¶

In order to create TFRecord files, the dataset has to be arranged into subdirectories, where each one represents the class. In our case, the dataset is already arranged in this way.

def load_tf(split_name): 
    with tf.Graph().as_default() as graph:
        tf.logging.set_verbosity(tf.logging.INFO)

        dataset = get_split(split_name=split_name, 
                            dataset_dir=tf_dir, 
                            file_pattern=file_pattern, 
                            file_pattern_for_counting=tfrecord_filename, 
                            items_to_descriptions=None)
        image, image_raw, label, image_name = load_batch(
            dataset=dataset,
            batch_size=1, 
            MODEL=MODEL,
            height=image_size,
            width=image_size,
            is_training=False)

        sv = tf.train.Supervisor(logdir='../log/tf_maker')

        with sv.managed_session() as sess:
            for i in range(dataset.num_samples):
                images, images_raw, labels, image_names = sess.run([image, image_raw, label, image_name])
                img, img_raw, lbl, img_n = images[0], images_raw[0], labels[0], image_names[0]
                print(images.shape)
                print(images_raw.shape)
                print(lbl)
                print(img_n)
                imshow(img_raw)
                break
